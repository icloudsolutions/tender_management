Project Templates
=================
This module will help you to create projects and tasks from templates, with advanced CRM integration for automatic project creation.

Features
========
* Create projects from project templates
* Multiple levels of tasks and sub tasks from template
* **CRM Integration**: Auto-create projects when opportunities reach specific stages
* Configure project templates to trigger on specific CRM stages
* Link projects to CRM opportunities for full traceability
* View all projects created from a CRM opportunity
* Support for multiple templates per CRM stage
* Automatic task creation from templates
* Settings to auto-create project templates from CRM stages

Configuration
=============
CRM Integration (Optional)
---------------------------
1. Go to Project → Configuration → Project Templates
2. Create or edit a template
3. Open the "CRM Integration" tab
4. Enable "Auto-create Project from CRM"
5. Select the CRM stage that should trigger project creation

Settings
--------
Go to Settings → Project Templates to enable:
* Auto-create Project Template from CRM Stage: Automatically creates project templates when CRM stages are created
